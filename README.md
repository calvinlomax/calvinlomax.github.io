# Personal Portfolio (GitHub Pages)

This repository is a clean, single-page portfolio starter designed for GitHub Pages.

## Included Criteria

- Clear positioning in the hero headline and summary
- 4 strong projects (expandable to 3-6)
- Quantified results for each project
- Clean responsive layout (desktop + mobile)
- GitHub links for each project
- Resume PDF link (`assets/resume/Calvin_Lomax_Resume.pdf`)
- Contact section with email + form

## Quick Customization

1. Update your name, title, and metrics in `index.html`.
2. Replace project descriptions, outcomes, and GitHub URLs.
3. Update contact details in `index.html` and `script.js` if needed.
4. Replace `assets/resume/Calvin_Lomax_Resume.pdf` whenever you have a new resume version.

## Publish to GitHub Pages

1. Push this repository to GitHub.
2. In GitHub: `Settings -> Pages`.
3. Under `Build and deployment`, set:
   - Source: `Deploy from a branch`
   - Branch: `main` and `/ (root)`
4. Save and wait for deployment.

Your site will be available at: `https://<your-username>.github.io/<repo-name>/`
